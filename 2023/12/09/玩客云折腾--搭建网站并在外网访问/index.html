玩客云折腾--搭建网站并在外网访问

John Doe2023-12-092023-12-10

前言

随着IPv4资源的枯竭，IPv6开始走入我们的视野，许多运营商开始为用户分配IPv6（默认公网），这就为我们的“折腾”带来很大便利

折腾的前提是你的玩客云（其他的也行，只要是Linux系统均可）已经刷入了Armbian系统，并已经正确部署了你的网站，还要求你的网络要接入了IPv6,想知道自己是否有，可以到 这个网站 查看，如果有，你会看到类似这样的界面

那么你可以开始折腾了！否则，请检查路由器设置或向运营商咨询

开始

方法一

首先，来到 每步 ，点击“我同意”，来到域名注册页面，根据相关提示填写即可。
然后来到玩客云终端，使用 apt install python3来安装python，我们需要用python来向服务器提交设置域名解析。
将一下代码保存为ipv6.py，将域名和密码替换成你自己的，保存后输入python3 ipv6.py运行，若看到chenggong，则域名解析成功，控制台相应输出当前IPv6地址，这时到一台 支持IPv6的设备 上访问你的域名即可。

如果报错，使用apt install python3-pip然后输入pip install requests即可

方法二

1.

来到 dynv6 ，注册账号，申请域名，添加自己玩客云的IPv6记录。

注册过程中涉及到Google的recaptcha验证码，需要有科学上网环境。

2.

来到玩客云终端，运行以下命令：

如果下载失败，可以将第一行命令换成wget https://mirror.ghproxy.com/https://github.com/jeessy2/ddns-go/releases/download/v5.6.6/ddns-go_5.6.6_linux_armv7.tar.gz即可。

3.

接着再dynv6的界面中点击你的账户，再点击keys–Add HTTP Token，然后填写名称，点击Generate HTTP Token便成功获取到token了。

4.

访问玩客云IP:9876，新建一个，选择callback，URL处填http://dynv6.com/api/update?zone=你申请的域名&token=你刚申请到的token&ipv6=auto`即可实现动态域名解析。
